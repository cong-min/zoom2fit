####使用须知

###开始使用
在`body`标签前插入代码：
```
<script src="jquery.min.js"></script>
<script src="zoom2fitJq.js"></script>
```
其中`jquery.min.js`及`zoom2fitJq.js`分别为`jquery`与`zoom2fit`的`js`文件地址
